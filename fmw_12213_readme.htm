1.1 About This Release

Oracle Fusion Middleware 12c (12.2.1.3.0) is the latest release of the Fusion Middleware 12c (12.2.1) software. It includes new features and improvements to the previously released 12c (12.2.1.2.0) products.

1.2 How to Install or Upgrade to This Release

For installation, upgrade, and patching information for this software, refer to the Oracle Fusion Middleware 12.2.1.3 documentation on the Oracle Help Center.

In particular, refer to the following sections of the Help Center, as you plan the installation:

  • If you are installing Oracle Fusion Middleware for the first time, refer to the Install and Patch page, which provides links to the installation documentation for all the Fusion Middleware products.

  • If you are upgrading from Oracle Fusion Middleware 12c (12.2.1.0.0) or from an earlier release, refer to the Upgrade page, which provides links to upgrade documentation for all the Fusion Middleware products.

In addition, refer to the following My Oracle Support note:

  • Oracle Fusion Middleware 12c Release 2 (12.2) Announcements (Doc ID 2067900.1).

My Oracle Support is available at http://support.oracle.com.
